SeatArranger
============

Array students' seat for teacher in school. [学生排座]献给教师们

功能列表
--------

* 设置课桌
  + 横向组为数值
  + 纵向组可以进行小组的划分

* 设置学生名单
  + 读取本地已保存的名单
  + 添加、修改、删除
  + 将学生转入教室等待排座

* 项目方案
  + 保存、读取方案
  + 打印当前分配好的座位表

* 其他
  + 分好的座位可以进行整体左右前后移动
